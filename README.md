# Sophia Lin - Personal Portfolio Website

This is my personal portfolio website！✨
(Created as the final SI 539 - WN2025 project at the University of Michigan)

## Overview
The site is fully hand-coded using:
- **HTML** for structure
- **CSS** for design and responsiveness
- **JavaScript** for interactivity

It includes three main pages:
- **Home**: Introduction and services overview
- **Portfolio**: Selected projects and case studies
- **About Me**: Personal journey and resume download

## Key Features
- Responsive design for mobile, tablet, and desktop
- Accessible navigation with "Skip to Main Content" link
- Interactive LinkedIn button and subscription form
- Auto-updating copyright year

## How to View
You can open `index.html` in a web browser to view the homepage and navigate the full site.
